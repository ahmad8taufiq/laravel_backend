### Host
http://localhost:8000/api

### Endpoint

- Create: POST /products
  ```
  {
    "code": "",
    "name": "",
    "category": "",
    "unit": "",
    "price: 0
  }
  ```

- Read All: GET /products

- Read One: GET /products/{id}

- Update: PUT /products/{id}
  ```
  {
    "code": "",
    "name": "",
    "category": "",
    "unit": "",
    "price: 0
  }
  ```

- Delete: DELETE /products/{id}
